Ví dụ dưới dùng httpie để test

LOGIN

  • /api/login (VD: http PUT http://localhost:5000/api/login username=alice password=susan1234)
Nếu đúng tài khoản mật khẩu, trả về toàn bộ thông tin người dùng như /api/user/1 . Dùng thông tin này để lấy user_id và bỏ vào phần thông tin của tôi.

GET API list

  • /api/users
  • /api/user/1
  • /api/rauculist Lọc rau cu theo raucu_type hoac shop_id: VD http GET http://localhost:5000/api/rauculist raucu_type=rau
  • /api/raucu/1
  • /api/notifications
  • /api/notification/1
  • /api/receipts Lọc hóa đơn theo user_id: VD http GET http://localhost:5000/api/receipt user_id=1
  • /api/receiptlist
  • /api/receipt/1
  • /api/carts
  • /api/bookmarks
  • /api/reviews Lọc review theo raucu_id hoac shop_id: VD http GET http://localhost:5000/api/rauculist raucu_id=11
  • /api/review/1
  • /api/shop/1
  • /api/shops

POST API list (for creating new entry in database)

  • /api/users (VD http POST http://localhost:5000/api/users username=alice password=dog email=alice@example.com)
  • /api/bookmarks (VD http POST http://localhost:5000/api/bookmarks user_id=1 raucu_id=2)
  • /api/resetpassword (VD http POST http://localhost:5000/api/resetpassword email=alice@example.com)
  • /api/reviews (VD http POST http://localhost:5000/api/reviews user_id=1 raucu_id=1 stars=1 comments="This is so badddddd")
  • /api/receipts (VD http POST localhost:5000/api/receipts user_id=1 shipping_cost=3333 shipping_addr=12345 total_price=1234)
  • /api/receiptlist (VD http POST localhost:5000/api/receiptlist receipt_id=1 raucu_id=3333 quantity=10)
  • /api/carts (VD http POST http://localhost:5000/api/carts user_id=1 raucu_id=2 quantity=10)

PUT API list (for updating values in database)

  • /api/users/1 (VD: http PUT http://localhost:5000/api/users/1 phone_no=123456)
  • /api/changepassword (VD: http PUT http://localhost:5000/api/changepassword id=3 password=Sudan2245)
  • /api/carts (VD: http PUT http://localhost:5000/api/carts user_id=1 raucu_id=2 quantity=2)
  • /api/receipts (VD: http PUT http://localhost:5000/api/receipts id=1 order_status=dahuy)
  • /api/reviews (VD http PUT http://localhost:5000/api/reviews user_id=1 raucu_id=1 stars=1 comments="This is so badddddd")

DELETE API list (for deleting row in database)

  • /api/bookmarks (VD: http DELETE http://localhost:5000/api/bookmarks user_id=1 raucu_id=1)
  • /api/carts (VD: http DELETE http://localhost:5000/api/bookmarks user_id=1 raucu_id=1)
